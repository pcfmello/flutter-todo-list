# TODO LIST APP
A todo list mobile application created by Flutter to training my skills.

## This app was used:
* Future Type;
* Path Provider;
* Material UI;
* Text Editing Controller;
* IO;
* Async/await;
* JSON convert;
* Dismissible;
* CheckboxListTile;
* Snackbar;

## Minimum Requisits  
You must have [Flutter environment installed](https://flutter.dev/docs/get-started/install)
